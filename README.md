# Aarika's Maths Dashboard 🌟

Interactive revision dashboard for **Aarika** — Std 3, Semester 2, Maharashtra SSC Board (2025-26).

## Topics Covered

| Topic | Contents |
|-------|----------|
| ⏰ Measurement of Time | Live clock, AM/PM, hour↔minute conversions, calculator |
| 📅 The Calendar | March 2026 calendar, 12 months, week conversions |
| 🍕 Fractions | Pizza visual, fraction bars, compare, add & subtract |
| 📊 Handling Data | Bar charts, tally marks, pictographs |

## How to Use

Open `Aarika_Maths_Dashboard.html` in any web browser — no internet required after first load.

## Features
- 🕐 Live analog clock with interactive demo times
- 🍕 Clickable pizza fraction visualizer
- 📊 Animated bar charts & pictographs
- 🧮 Hours → Minutes live calculator
- 🧠 Mini quiz for each topic
- 🗓️ March 2026 calendar with today highlighted
